# Host Kit

### An easy to use Open Source Project

#### Summary :

- On Host Kit you will find many different projects associated to **ease of installation**, **fully customisable project** and **cross-platform compatibility**.


Start of the project : *23rd of April 2026*


The goal of Host Kit is to offer to anyone, who self hosts or wants privacy and functionality for free (Open Source and contribution based resources), free tools and software to make using other self hosted tools and software easy and customisable. In addition to which, Host Kit offers a variety of Open Source projects to make self hosting, and decentralized software and applications. In order to entirely REMOVE advertisements and profiling from taking your attention span (written in 2026 after what "A.I." has done to massive enterprises).
I want for the web to be helpful and serve meaningful resources to users, instead of SLOP.
This is why I believe that both centralized and decentralized web are useful. One for source and the other for utilities, such as file transfer (File Pizza - a great P2P file sending tool), messaging (Keet - I am trying it out as of early 2026) and more.
My question is : "Why is the web so centralized?" - to which I respond : "Because people let it be so."

I will read your suggestions carefully.
